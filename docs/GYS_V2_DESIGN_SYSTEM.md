# GYS-V2 PROTOCOL: DESIGN SYSTEM ARCHITECTURE

## 1. Core DNA (Color Palette)
- **Background (Deepest Navy):** `#020617` (The void, authority, and depth).
- **Primary Accent (Electric Blue):** `#3B82F6` (Speed, energy, and innovation).
- **Secondary Accent (AI Logic):** `#10B981` (Success, calculation, and stability).
- **Surface (Logic Glass):** `rgba(30, 41, 59, 0.5)` with `12px backdrop-blur`.
- **Text (High Contrast):** `#F8FAFC` (Primary), `rgba(248, 250, 252, 0.4)` (Muted/Mnemonic).

## 2. Typography (The System Voice)
- **English:** `Inter` (Geometric Sans-Serif).
  - Headings: Bold/Extra-Bold, Tracking: -0.02em.
  - Body: Light/Medium, Tracking: 0.01em.
- **Arabic:** `IBM Plex Sans Arabic`.
  - Geometric curves, digital clarity, no traditional calligraphic flares.

## 3. The Logic UI (Interactions)
- **Border Radius:** `1rem` (16px) standard for AI components.
- **Transitions:** `200ms cubic-bezier(0.4, 0, 0.2, 1)` (Instant response feel).
- **Glow Protocol:** 
  - Outer Glow: `0 0 40px rgba(59, 130, 246, 0.3)`.
  - Focus Pulse: Cyan-Blue radial pulsation.

## 4. Grid Protocol
- **Setup:** 12-column strict grid.
- **Spacing:** Fluid modular scale (Factor: 4px).
