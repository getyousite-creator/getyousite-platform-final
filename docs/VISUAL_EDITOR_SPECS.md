# مواصفات المحرر البصري السيادي (Visual Editor Specs)

## 1. التعديل المباشر (Direct In-Place Editing)
- **النصوص:** أي نص تضغط عليه يتحول فوراً إلى حقل إدخال (ContentEditable).
- **الصور:** الضغط على الصورة يفتح واجهة تبديل ذكية (تكامل مع Unsplash أو توليد بالذكاء الاصطناعي).

## 2. مساعد التعديل الذكي (AI Tweaker Panel)
بدلاً من التحكم اليدوي المعقد، نستخدم "النافذة السحرية":
- **المدخل:** "اجعل هذا القسم أكثر حيوية" أو "غير نبرة النص لتكون أكثر رسمية".
- **الفعل:** يقوم النظام بتعديل (Design Tokens) لهذا القسم تحديداً (تغيير الألوان، حجم الخط، نغمة الكتابة) مع عرض معاينة فورية.

## 3. لوحة التحكم الهندسية (The Logical Sidebar)
تظهر عند اختيار أي مكون، وتتيح التحكم في:
- **البنية (Structure):** إضافة/حذف عناصر داخل المكون.
- **النمط (Themes):** التبديل بين أنماط فرعية لنفس المكون (مثلاً: Hero مع فيديو خلفية مقابل Hero مع تدرج لوني).

## 4. نظام الحصانة والرجوع (Immunity & Undo)
- سجل حركات كامل (Unlimited Undo/Redo).
- **Auto-Save Protocol:** حفظ تلقائي لكل حركة في مسودة مؤقتة قبل الاعتماد النهائي (Publish).

## 5. واجهة المعاينة (Perspective View)
أزرار سريعة للتبديل بين (Desktop, Tablet, Mobile) لضمان أن التصميم "المستجيب" يعمل بجمالية في كل الأبعاد.
